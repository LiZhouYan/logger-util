日志工具：日志等级调整等
我个人参考对美团的日志调整项目做了相应的调整，适应自己的当前情况，在下一个文章做具体的介绍。

   具体使用方式
　　引入jar包，这个包是个人的，可以根据个人喜爱，随意操作，放在maven私有仓库中

        <dependency>
            <groupId>com.ninebot.apr</groupId>
            <artifactId>logger</artifactId>
            <version>1.0-SNAPSHOT</version>
        </dependency>
